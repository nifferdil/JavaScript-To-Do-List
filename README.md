# To Do List

##### _September 21 2015_

#### By **Jennifer Morkunas**

## Description

Add tasks and mark them complete by adding them to a "completed" list.

## Setup

* Clone this repository.
* Open the index.html file in your browser.
* Start adding input via forms provided.

## Technologies Used

* JavaScript
* jQuery
* Mocha
* Chai
